# 🌀 Naruto UHC Plugin
Plugins Naruto UHC by fazeplay (Velyq)

Ce plugin implémente le mode de jeu complet **Naruto UHC** sur votre serveur Minecraft.

## 📌 Version Supportée

* **Version de Minecraft :** `1.8` / `1.8.8`
* **Version de Java requise :** `Java 17` (requis pour le fonctionnement du plugin de base et de ses dépendances modernes sur cette version de Minecraft).

---

## 🔌 Dépendances Obligatoires

Pour que le plugin fonctionne correctement sans crash au démarrage, vous devez impérativement installer les deux dépendances ci-dessous dans votre dossier `plugins/`.

### 1. ProtocolLib (Version Spéciale Java 17 / Minecraft 1.8)
La version standard de ProtocolLib pour la 1.8 ne prend pas en charge Java 17. Vous devez utiliser ce build spécifique modifié pour assurer la compatibilité :
* 👉 **Télécharger :** [ProtocolLib v5.1.0-SNAPSHOT (Compatible 1.8 & Java 17)](https://github.com/dmulloy2/ProtocolLib/releases)

### 2. OpenAudioMc
Requis pour la gestion du chat de proximité
* 👉 **Télécharger :** [OpenAudioMc (Spigot/Paper)](https://spigotmc.org/resources/openaudiomc-realtime-audio-proshout-voice-chat.30691/)

### 2. OpenAudioMc
Requis pour la gestion des uhc.
* 👉 **Télécharger :** uhc-api.jar

---

## ⚙️ Installation

1. Arrêtez votre serveur Minecraft.
2. Glissez le fichier `.jar` de ce plugin ainsi que les deux dépendances (`ProtocolLib` et `OpenAudioMc` et `uhc-api.jar` ) dans le dossier `plugins/`.
3. Démarrez votre serveur et amusez-vous ;).
